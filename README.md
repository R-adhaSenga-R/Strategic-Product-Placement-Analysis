Strategic Product Placement Analysis

This project analyzes retail sales data to understand how product placement strategies influence sales performance, profitability, and customer purchasing patterns.

Using Tableau Public, interactive dashboards and stories were created to visualize key insights from the dataset.
A Flask-based web interface was also developed to embed the Tableau dashboard and story into a responsive UI.

Project Objective

The main goal of this project is to analyze retail product placement strategies and identify patterns that impact:

Sales performance

Profit generation

Customer purchasing behavior

Regional product demand

Through interactive visualizations, businesses can make data-driven decisions to improve product positioning and maximize revenue.

Dataset

The dataset used in this project contains retail sales data including the following attributes:

Product Category

Sales

Discount

Region

Profit

The dataset was analyzed to identify relationships between product placement and sales performance.

Key Insights

The analysis provides insights such as:

High-performing product categories by region

Impact of discounts on profitability

Sales trends across different product segments

Profit distribution across regions

These insights help businesses optimize product placement and marketing strategies.

Web Application (Flask Integration)

To make the analysis more interactive, a Flask web application was developed that embeds the Tableau dashboard and story within a web interface.
